# Contribuir

Este es un proyecto personal de portfolio, pero el repositorio está organizado para poder revisarse, ejecutarse y extenderse con facilidad.

## Flujo recomendado

1. Crear una rama desde `main`.
2. Hacer cambios pequeños y verificables.
3. Ejecutar pruebas del backend.
4. Validar build o TypeScript del frontend.
5. Abrir pull request con una descripción clara.

## Comandos útiles

Backend:

```powershell
cd apps\api
pytest
```

Frontend:

```powershell
cd apps\web
npm run build
```
