# Guion de demo

Duración sugerida: 60 a 90 segundos.

## 1. Presentación

Mostrar la pantalla inicial y explicar:

```text
Multimedia AgentOps Studio transforma briefs creativos en campañas multimedia completas usando agentes de IA especializados.
```

## 2. Brief

Cargar un ejemplo o escribir una marca propia con audiencia, tono, restricciones y canales.

## 3. Generación

Ejecutar la campaña y mostrar el estado del proveedor.

## 4. Timeline

Mostrar el progreso del flujo de agentes y explicar que cada agente produce una parte distinta de la campaña.

## 5. Resultado

Mostrar:

- resumen ejecutivo;
- estrategia;
- copy;
- storyboard;
- dirección visual;
- plan de ejecución;
- KPIs;
- evaluación.

## 6. Historial y exportación

Mostrar que la campaña queda guardada, puede renombrarse, fijarse, eliminarse y exportarse.

## 7. Cierre

Explicar brevemente el stack:

```text
Next.js, TypeScript, Tailwind, FastAPI, Python, SQLite y Groq API.
```
